# Haus Designr AI

This is the source code for Haus Designr AI, an entry in the dynamic demo competition at Alternative Party 2024.

It contains some spaghetti at the moment, which I will straighten up soon after the event. (Published early due to competition deadline.)

It is made in Unity, and Assets made by other people may not be uploaded to GitHub (even if free), so the project contains only the code and some self-made assets, but not all assets required to run the demo properly.

The soundtrack file is too large for GitHub, and will be available separately on SoundCloud.
